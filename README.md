# 🎭 Living Anime Space

3DアニメキャラクターとAIでリアルタイム会話ができるウェブアプリケーションです。

## ✨ 機能

- 🤖 **AI会話** - ChatGPT / Gemini / Grok Voice でリアルタイム会話
- 🎭 **3Dキャラクター** - VRMモデルの表示・操作・ドラッグ＆ドロップ読み込み
- 💃 **モーション** - 感情に連動した自動モーション再生
- 🔊 **音声合成** - OpenAI TTS / Google TTS による読み上げ
- 🎥 **AIカメラ** - シネマティックカメラワーク
- 📱 **PC・スマホ対応**

## 🚀 使い方

1. サイトにアクセス
2. 画面左上の **「🔑 API設定」** ボタンをクリック
3. 使いたいサービスのAPIキーを入力:
   - **OpenAI API Key** (ChatGPT / TTS) → [取得先](https://platform.openai.com/api-keys)
   - **Google API Key** (Gemini / Google TTS) → [取得先](https://aistudio.google.com/)
   - **Grok API Key** (xAI Voice) → [取得先](https://console.x.ai/)
4. チャット欄からメッセージを送信！

> 💡 APIキーはブラウザのlocalStorageに保存され、サーバーには一切送信されません。

## 🔧 自分のVRMモデルを使う

画面にVRMファイルをドラッグ＆ドロップするだけ！
